Football Analytics — Data Import

One-click import of all CSV seasons into Firebase
How this works: This page downloads football match data directly from football-data.co.uk (free, no API key) and saves it to your Firebase Firestore. Once imported, your main app loads all data instantly from Firebase — no more proxy issues, works offline, syncs across all your devices. You only need to re-import the current season after each matchday (takes ~5 seconds).
Step 1 — Sign in to Firebase
Sign in with the same Google account you use on the main app.
Step 2 — Import seasons into Firebase
Green = already in Firebase. Click Import to fetch from football-data.co.uk and save. Use Import All Missing to do everything in one go.
